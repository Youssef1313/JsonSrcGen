# JsonSG
Json library that uses .NET 5 c# Source Generators

Note: Requires the .Net 5 Preview to run. Mono platforms do not have support for the .Net 5 Source Generators yet.

# Supported Types
Json object with the following properties:
* int
* int?
* uint
* uint?
* ushort
* ushort?
* short
* short?
* byte
* byte?
* long
* long?
* ulong
* ulong?
* boolean
* boolean?
* string
